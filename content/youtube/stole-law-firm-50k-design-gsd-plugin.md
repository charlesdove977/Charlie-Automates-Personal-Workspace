## Key Points

2. **The Mistake:** People use Frontend Design skill without planning/research
3. **The Solution:** Research → Plan → Build (with proper context)
4. **The Secret Sauce:** Claude in Browser to spy on competitors + Frontend Design skill

## Workflow to Demo

1. Show current generic CaseGat.ai UI (the "before")
2. Use Claude to research top attorney websites
3. Open Claude in Browser to analyze premium law firm sites
4. Extract design patterns, colors, layouts
5. Create a gameplan/brief for the UI rebuild
6. Use Frontend Design skill WITH the research context
7. Show the polished result (the "after")

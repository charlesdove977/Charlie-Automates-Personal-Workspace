# TwinCast Runner

Start the TwinCast application (backend + frontend servers).

## Instructions

Run the TwinCast startup script to launch both servers:

```bash
cd /Users/user/Downloads/Development\ Charlie\ 2/Charlieautomates/apps/twin-cast && ./rtc.sh
```

This will start:
- **Frontend**: http://localhost:3000
- **Backend**: http://localhost:8000
- **API Docs**: http://localhost:8000/docs

Report the URLs to the user when the servers are running.

  <action>
1. Create virtual environment (if not exists):
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Create a temporary test script `test_sdk.py` to verify SDK installation:
   ```python
   import anyio
   from claude_agent_sdk import query, ClaudeAgentOptions

   async def main():
       print("SDK imported successfully!")
       print(f"ClaudeAgentOptions available: {ClaudeAgentOptions}")
       # Don't actually run a query (costs money), just verify import works

   if __name__ == "__main__":
       anyio.run(main)
   ```

4. Run the test:
   ```bash
   python test_sdk.py
   ```

5. Clean up test file after verification:
   ```bash
   rm test_sdk.py
   ```

Do NOT run actual queries against the API in this task - just verify imports work.
  </action>
  <verify>
